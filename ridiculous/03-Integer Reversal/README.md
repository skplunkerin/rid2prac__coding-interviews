# Integer Reversal

**categories:** [Vague, Overly Simplified, or Zero Context](#vague-overly-simplified-or-zero-context)

---

## Problem:

Given an integer, return an integer that is the reverse ordering of numbers.

### Examples:

| input | expected output |
| ----- | --------------- |
| 15    | 51              |
| 981   | 189             |
| 500   | 5               |
| -15   | -51             |
| -90   | -9              |

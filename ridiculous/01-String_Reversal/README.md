# String Reversal

_**categories: [vague/overly simplified](https://github.com/skplunkerin/rid2prac__coding-interviews#vague-or-zero-context)**_

---

## Problem:

Given a string, return a new string with the order of characters reversed.

## Examples:

- `"apple"` -> `"elppa"`
- `"hello"` -> `"olleh"`
- `"Greetings!"` -> `"!sgniteerG"`
